Selenium Grid2

Mewes Kochheim

Introduction

"Selenium-Grid allows you run your tests on different machines against different browsers in parallel."

History

  • currently in version 2.43.1
  • part of the Selenium server
  • successor of Selenium RC
  • backwards compatible

Selenium server

  • Selenium core
  • Selenium hub
  • Selenium node

Preparations

Selenium IDE

"Selenium IDE is an integrated development environment for Selenium scripts. It is implemented as a Firefox extension, and allows you to record, edit, and debug tests."

run as part of the build process

and/or

run continuously

Selenium Hub

$ java -jar selenium-server-standalone-2.43.1.jar -role hub

Nodes

...are annoying

Supported Operating Systems

  • Windows 7
  • Windows 8
  • Windows 8.1
  • OS X (latest)
  • Ubuntu
  • Android (Chrome only)
  • ChromeOS (Chrome only, duh)

Supported Browsers

  • Firefox (latest, previous, latest ESR and previous ESR)
  • Internet Explorer (6, 7, 8, 9 and 10)
  • Safari (5.1+)
  • Opera (12.x)
  • Chrome (latest stable, beta, and dev, plus previous stable)
All browsers except Firefox require external driver applications!

Example

set PATH=%PATH%;C:\Selenium
start cmd /k java ^
  -jar C:\Selenium\selenium-server-standalone-2.40.0.jar ^
  -role node ^
  -hub http://192.168.1.1:4444/grid/register
start cmd /k IEDriverServer.exe --port=9516
start cmd /k chromedriver.exe
